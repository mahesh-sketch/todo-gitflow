# todo-gitflow (training repo)

A tiny JavaScript project to practice Git workflows with `develop`, feature branches, rebasing, conflicts, and PRs.

## Files
- `src/todo.js` – the single code file we will modify to create conflicts and features.
- `.github/PULL_REQUEST_TEMPLATE.md` – a simple PR template.
- `.github/CODEOWNERS` – example owners (tweak for your org).
- `.gitignore`, `package.json` – basic Node setup (optional).

Use it with the step-by-step guide in ChatGPT.
